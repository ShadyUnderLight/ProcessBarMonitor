## ProcessBarMonitor v1.0.0

First public packaged release of ProcessBarMonitor for macOS.

### Highlights
- Live menu bar overview for CPU, memory, and thermal state
- Top apps by CPU usage
- Top apps by memory usage
- Manual refresh, search, and row count controls
- Launch at login toggle
- Improved Quit behavior for the menu bar app

- ProcessBarMonitor-v1.0.0-macOS.zip

### Notes
- CPU temperature is best-effort and may fall back to thermal state when no supported helper tool is available.
- This release ships as a zipped macOS `.app` bundle.
